# 图书阅览系统 - 前端

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 环境变量

创建 `.env` 文件：

```
VITE_API_BASE_URL=http://localhost:5000
```

## 功能

- 📚 图书分类浏览（文学、金融、科技、历史、艺术）
- 📖 在线阅读（PDF、EPUB）
- 🔖 书签和笔记功能
- 🔍 图书搜索和筛选

## 技术栈

- Vue 3
- TypeScript
- Vue Router
- Pinia
- Axios
- EPUB.js
